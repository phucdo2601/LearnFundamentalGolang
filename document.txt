	/*
	Synchronizing in go:
		- Event: By Default, the main goroutine does not wait for other goroutines

		-Waitgroup:
			+ wait for the launched goroutine to finish
			+ Package "sync" provides basic Synchronization functionality
			+ Add:
				+ Sets the number of goroutines to wait for (increase the counter by the provided number)
			+ Wait:
				+ Blocks util the Waitgroup counter is 0

			+ Done:
				+ Decrements the Waitgroup counter by 1 => So this is called by the goroutine to indicate that it's finished

	*/
